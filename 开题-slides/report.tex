\begin{frame}
\begin{block}{}
Advantage:\\
增量式，无需指定簇个数(VS K-means)\\
对噪声不敏感\\
能发现任意形状的簇\\
\end{block}
\begin{alertblock}{}
Disadvantage:\\
数据量较大时，要求较大的内存支持和运算时间\\
空间密度不均匀时，聚类质量较差
\end{alertblock}

\end{frame}

\begin{frame}

        \item 经验
        \item 评价函数、学习
        \item 核聚类(非线性变换，分布均匀)
      \end{itemize}
并行化\\
\end{frame}
